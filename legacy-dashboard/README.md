## 📖 Descripción

**R-C-T-E-E Pro Dashboard** es una herramienta profesional de generación de prompts empresariales basada en el framework **R-C-T-E-E** (Rol, Contexto, Tarea, Especificaciones, Ejemplos). Diseñada para consultores, ingenieros de prompts y empresas que buscan maximizar el rendimiento de modelos de IA como **Llama 3.3**, **GPT-4** y **Claude**.

Incluye un **panel de control moderno** con sidebar, métricas en tiempo real, chatbot IA integrado y más de **60 plantillas especializadas** listas para usar.
